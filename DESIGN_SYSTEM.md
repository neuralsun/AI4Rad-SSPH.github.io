# AI4Rad Lab 视觉设计系统 v1.0

> 基于上海交通大学医学院附属第六人民医院（SSPH）视觉识别、团队 Beamer 模板、AI 大模型服务平台风格，以及团队公开发表论文主题整理而成。
> 
> ⚠️ 保密说明：本文件仅使用**已公开**的医院视觉资产、模板样式和已发表论文信息，不包含在研项目细节或未公开研究方向。

---

## 1. 设计原则

### 1.1 核心定位

AI4Rad Lab 是上海市第六人民医院放射介入科下属的医学影像人工智能研究团队。网站视觉应同时体现：

- **医院公立属性**：稳重、专业、可信赖
- **学术前沿感**：清晰、现代、数据驱动
- **临床转化导向**：从算法到临床应用的完整链路

### 1.2 风格关键词

| 关键词 | 说明 |
|--------|------|
| 白底清新 | 与医院信纸/汇报模板一致，以白色为底，用色块和线条做结构 |
| 蓝青绿主调 | 医院 VI 主色，传达冷静、专业、医疗科技感 |
| 对角几何 | Beamer 封面页使用的三角形/斜切元素，作为品牌记忆点 |
| 低饱水印 | 右下角建筑线稿水印，10-15% 透明度，强化医院归属感 |
| 信息密度高 | 学术团队网站，优先展示成员、论文、项目，避免营销化大留白 |

## 5. 布局与组件

### 5.1 页面结构

```
[固定导航栏]  Logo + 导航链接 + 语言切换 + 深色模式
[Hero 区]     医院联合 Logo + Lab 名称 + 一句话定位 + CTA
[数据条]      成员数 / 方向数 / 论文数 / 合作院校
[研究方向]    4 张公开方向卡片
[最新动态]    时间线式新闻
[团队入口]    核心成员预览 + 查看全部按钮
[加入我们]    招生/合作信息
[页脚]        机构信息 + 链接
```

### 5.2 品牌几何元素

从 Beamer 封面提取：

- **右下角大三角形**：Primary color，10% 透明度，作为页面背景装饰
- **左上角小三角形**：Accent color，用于 Hero 区点缀
- **短粗分隔线**：Primary color，4px 高，用于区块标题下方

### 5.3 卡片组件

- 白底 + 1px Line 边框
- 圆角 16–20px
- hover 时轻微上移 + 阴影
- 图标使用 40×40px 圆角色块，背景为 Pale Field

## 6. 团队研究框架（来自飞书团队地图）

> 以下七大医学方向与四类技术能力来自团队内部公开的方向地图文档，属于团队层面的研究框架，不暴露具体在研项目细节。

### 6.1 七大医学方向

| 方向 | 一句话说明 |
|------|-----------|
| 泛血管影像与血流疾病 | 心脑、颈动脉、主动脉、下肢等血管结构、血流、灌注与临床事件 |
| 骨肌系统与骨定量 | 骨、关节、肌腱、韧带、脊柱、骨折与骨代谢相关影像问题 |
| 肿瘤影像与全身转移 | 以影像为入口的肿瘤诊断、分期、疗效、预后与病理匹配 |
| 慢病队列与长期随访 | 依托医院队列，挖掘多次影像检查中的疾病演变信息 |
| 影像教育与报告反馈 | AI 辅助报告书写、教学训练、质量控制与医生反馈机制 |
| 急诊影像与腹部疾病诊断 | 时间敏感型头颅、腹部、胸部急诊影像诊断 |
| 新影像技术与临床转化 | 光子计数 CT、能谱 CT、快速 MRI、4D flow 等新技术的临床应用 |

### 6.2 四类技术能力

| 方向 | 一句话说明 |
|------|-----------|
| 视觉基础模型 | 医学影像视觉表征与结构化理解（诊断、分割、检测、定量） |
| 多模态大语言模型 | 影像、报告、病史、知识库的多模态智能与临床推理 |
| 影像生成 | 面向临床价值的图像生成、修复、增强与跨模态合成 |
| 空间智能与物理仿真 | 三维建模、配准、血流/骨肌力学仿真与可解释定量分析 |

### 6.3 网站展示口径

首页「研究方向」区块当前展示**四类技术能力**作为团队技术支柱，并在引言中说明这些能力服务于七大临床医学方向。这种表述：

- 准确反映团队官方方向地图
- 不暴露个人项目分配和具体在研课题
- 区分团队工作与个人工作

### 6.4 团队工作 vs 个人工作的区分

以下为代表性论文的归属说明（供内部参考，网站不展开）：

- **BrainMIND**（*npj Digital Medicine* 2026）→ 团队工作
- **UniChest**（*IEEE TMI* 2024）、**FedDG-GA**（*CVPR* 2023）、**FACT**（*CVPR* 2021 / *Pattern Recognition* 2023）→ 张瑞鹏个人前期工作，非 AI4Rad Lab 团队项目
